Questionário sobre CSS (Folhas de Estilo em Cascata)

Perguntas:

1-O que significa CSS?

CSS significa "Cascading Style Sheets", que em português pode ser traduzido como .

2-Cite 2 características importantes quando se usa CSS?

Separar o estilo do conteúdo e Cascata

3-Quais as possíveis formas de vincular o CSS em um documento HTML? Cite e exemplifique

Vinculação Externa: O CSS é colocado em um arquivo separado e vinculado ao documento HTML usando a tag no cabeçalho do documento. O atributo href especifica o caminho para o arquivo CSS.Vinculação Interna: O CSS é incluído diretamente no cabeçalho do documento HTML usando a tag style. O CSS é inserido entre as tags style e /style.Inline: O CSS é aplicado diretamente em um elemento HTML específico usando o atributo style.

4-Na definição do CSS o que representa a criação de uma regra no seletor?

Em CSS, a criação de uma regra no seletor é o processo de definir o estilo que deve ser aplicado a um ou mais elementos HTML em uma página da web. Uma regra é composta por um seletor e um bloco de declarações de estilo, que especifica como o(s) elemento(s) devem ser estilizados. O seletor é usado para identificar o(s) elemento(s) HTML que devem ser estilizados, enquanto o bloco de declarações de estilo é usado para definir o estilo que deve ser aplicado.

5-Como se comenta uma linha na definição do arquivo CSS?

Para comentar uma linha em um arquivo CSS, é necessário adicionar /* no início do comentário e */ no final. Qualquer texto entre esses caracteres não será processado como parte do arquivo CSS. Comentários podem ser adicionados ao lado das declarações de estilo para ajudar a documentar o código.

6-Quais são os delimitadores de início e fim na definição de uma regra do CSS?

Os delimitadores de início e fim em uma regra do CSS são representados pelas chaves { e } respectivamente. O seletor é usado para identificar o elemento HTML que será estilizado, enquanto o bloco de declarações de estilo contido entre as chaves define como o elemento deve ser estilizado.

7-Quando o ; será utilizado?

O ponto e vírgula (;) é utilizado para separar cada declaração de estilo em uma regra CSS. Cada declaração inclui uma propriedade e um valor separados por dois pontos (:). O ponto e vírgula é usado para indicar o final de cada declaração de estilo e para separar as declarações subsequentes dentro do bloco de declarações de estilo.

8-O que são valores múltiplos para serem usados nas regras CSS?

Valores múltiplos em CSS permitem definir mais de um valor para uma propriedade em uma única declaração. Esses valores são separados por espaços em branco dentro da declaração de estilo e são aplicados na ordem em que aparecem. Um exemplo é a propriedade font-family, que pode ter vários valores para especificar uma lista de fontes a serem usadas em ordem de preferência.
